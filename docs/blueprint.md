# **App Name**: SportVerse

## Core Features:

- Dashboard Layout: Implements a responsive layout with a fixed header, sidebar, and main content area to display application name and key functionalities.
- User Interface Elements: Features utility icons (notifications, settings, user profile) in the header, a vertical menu with active state indication in the sidebar, a prominent page title, and stat cards within the main content.
- Two-Column Content: Arranges the main content area into a two-column layout, featuring a recent activity feed (icon, description, timestamp) on the left and upcoming events modules on the right.
- Live Data Fetching: Retrieves real-time sports stats and data to populate summary stat cards, the recent activity feed, and the upcoming events table.

## Style Guidelines:

- Primary color: Dark blue (#243A73) for a sense of authority and calmness.
- Background color: Light blue (#D2E3FC) to complement the primary color.
- Accent color: Orange (#E38B29) for CTAs and to highlight data.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines, 'Inter' (sans-serif) for body text.
- Use clean, modern icons from a consistent set.
- Full-width header, fixed vertical sidebar, and two-column content layout for optimal information hierarchy and easy navigation.
- Subtle transitions and animations to provide feedback on user interactions and improve user experience.
